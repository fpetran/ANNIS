QueryData annis.AnnisRunner.analyzeQuery ( String  annisQuery,
String  queryFunction 
) [private]

Does the setup for the QueryData object.

If the query function is "subgraph" or "sql_subgraph" the annisQuery string should contain space separated salt ids. In this case the annisQuery is not parsed and the QueryData#getAlternatives() method should return a List with dummy QueryNode entries. Instead of parsing the annisQuery it extracts the salt ids and put it into the extension's of QueryData.

Parameters:
annisQueryshould include a valid annis query
queryFunctionshould include a method name of AnnisRunner which starts with do.
Returns:
QueryData object, which contains a parsed annis query, the default context AnnisRunner#left and AnnisRunner#left values and the default AnnisRunner#limit and AnnisRunner#offset values
