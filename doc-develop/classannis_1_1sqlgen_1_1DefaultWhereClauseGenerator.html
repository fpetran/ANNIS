void annis.sqlgen.DefaultWhereClauseGenerator.notReflexive ( List< String >  conditions,
QueryNode  node,
QueryNode  target 
) [private]

Explicitly disallow reflexivity.

Can be used if the other conditions allow reflexivity but the operator not. It depends on the search conditions if two results are not equal.

For two nodes (including searches for token):
node IDs are different

If both nodes are an annotation:
node IDs are different or annotation namespace+name are different

For a node with and one without annotation condition:
always different
