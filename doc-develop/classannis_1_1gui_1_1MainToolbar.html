annis.gui.MainToolbar Class Reference

Detailed Description

The ANNIS main toolbar.

Handles login, showing the sidebar (if it exists), the screenshot making and some information windows.

void annis.gui.MainToolbar.addLoginButton ( ) [private]

Adds the login button + login text to the toolbar.

This is only happened, when the gui is not started via the kickstarter.

The Kickstarter overrides the "kickstarterEnvironment" context parameter and set it to "true", so the gui can detect, that is not necessary to offer a login button.
