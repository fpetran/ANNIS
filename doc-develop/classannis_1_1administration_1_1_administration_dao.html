void annis.administration.AdministrationDao.analyzeTextTable ( String  toplevelCorpusName)
private

Searches for textes which are empty or only contains whitespaces.

If that is the case the visualizer and no document visualizer are defined in the corpus properties file a new file is created and stores a new config which disables document browsing.

Parameters
corpusIDThe id of the corpus which texts are analyzed.
