static boolean annis.ql.parser.DNFTransformer.makeDNF ( LogicClause  node) [static, private]

Iteration step to transform a LogicClause into DNF.

In DNF all OR relations must be toplevel. Thus constructions like

  AND             AND
  / \      or     / \
 X  OR          OR   X
    / \        /  \
   Y   Z       Y  Z

are illegal and will be replaced with

       OR       
     /    \    
   AND    AND  
   / \    / \    
  X   Y  X   Z

according to the distributivity rule of Boolean Algebra. We assume that every input formula is already binary.

Only one transformation will be done in this function, repeat it in order to replace all illegal constructions.

Parameters:
nodeThe node to transform into DNF.
Returns:
True if already in DNF
