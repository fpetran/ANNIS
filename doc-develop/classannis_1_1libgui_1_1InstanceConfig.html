annis.libgui.InstanceConfig Class Reference

Detailed Description

Hold information about the configuration of a specific (sub-) instance of ANNIS.

Each physical installation (speak deployment) of ANNIS is able to have several instances which behave differently. This is meant to provide a more specialized presentation for different projects while still using only one ANNIS installation.
