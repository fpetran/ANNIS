annis.libgui.MatchedNodeColors Enum Reference

Detailed Description

Colors for matched nodes.

The names and color values correspond to the CSS standard.

These colors must always be synchronized with the "match-color" definitions in the annis.scss style sheet.

Author:
Thomas Krause <krauseto-berlin.>
