annis.libgui.visualizers.AbstractVisualizer< I extends Component > Class Reference

Detailed Description

Base class for all Visualizer.

This class sets some defaults, so you may implement the VisualizerPlugin#createComponent(annis.gui.visualizers.VisualizerInput) method for pure Vaadin component plugins.
